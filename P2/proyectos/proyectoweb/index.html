Sobre Mí

Mi nombre es Natalia Niebla, soy estudiante de la Universidad Tecnológica de Durango, actualmente estudio en la carrera de Tecnologías de Información donde se abordan materias de programación y el diseño web.

Las tecnologías de desarrollo web que actualmente utilizamos en los proyectos que se desarrollan son: HTML, para la maquetación, CSS hojas de estilo para el diseño, JavaScript programación del lado del cliente, PHP como programación del lado del servidor y como gestor de Base de Datos MySQL

Te puedes poner en contacto conmigo por e-mail: natalia_3141230020@gmail.com
